An AI-powered meeting assistant that converts meeting audio into text, cleans and processes the transcript, and generates structured meeting minutes and actionable task lists using IBM Watsonx LLM and OpenAI Whisper for speech-to-text.
The application includes an easy-to-use Gradio web interface.

Audio File
   ↓
Whisper Speech-to-Text
   ↓
Transcript Cleaning
   ↓
IBM Watsonx LLM (Granite)
   ↓
Meeting Minutes + Task List
   ↓
Text Display + Downloadable File
